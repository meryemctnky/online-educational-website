## Installation

For development :

```javascript
// Firstly install dependecies
npm install

//After we can run project , run dev gonna start on dev mode
npm run dev
```

For production:

```javascript
npm run build

//After we can run project , gonna start on production mode
npm start
```

Open http://localhost:3000 with your browser to see the result.
